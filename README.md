# Pic-in-Pic-Universal
Pic-in-picture that works everywere (almost)

Universal PiP Extension — Installation Guide
1. Download and unzip

Download pip-extension-v2.zip and extract it to a folder you'll keep on your computer (don't delete it later — the browser loads the extension directly from this folder).

2. Chrome, Brave, Edge, Opera, Vivaldi (Chromium-based browsers)

These all use the same steps, just a different address for the extensions page:

Browser	Extensions page
Brave	brave://extensions
Chrome	chrome://extensions
Edge	edge://extensions
Opera	opera://extensions
Vivaldi	vivaldi://extensions

Steps:

Paste the address above into the URL bar and hit Enter.
Turn on Developer mode (toggle usually in the top-right corner).
Click "Load unpacked".
Select the unzipped pip-extension-v2 folder (the one containing manifest.json).
The extension's icon (the photo) will appear in the toolbar. Click the puzzle-piece icon and pin it if it's hidden.
3. How to use it

Open any streaming site, start playing a video, then click the extension's icon. It'll force Picture-in-Picture mode, waiting for the video to load if needed.
